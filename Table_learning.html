üí° What is an HTML Table?

Think of an HTML table like a spreadsheet in Excel or a grid on paper. It has:
  • üìã Rows - Horizontal lines (left to right)
  • üìä Columns - Vertical lines (up and down)
  • üìù Cells - Boxes where rows and columns meet

üéÆ Interactive Learning

1Create the Table Container

<table border="1">
  ... table content goes here ...
</table>

The <table> tag is like drawing a box on your page where your table will live.

2Add a Title

<table>
  <caption>Dhaka College</caption>
</table>

The <caption> is like writing a title above your table.

3Create Rows

<tr>
  ... cells go here ...
</tr>

<tr> means "table row" - it's like drawing a horizontal line in your table.

4Add Headers and Data

<tr>
  <th>Roll</th>  <!-- Header (bold) -->
  <td>01</td>   <!-- Regular data -->
</tr>

<th> = Table Header (bold text)
<td> = Table Data (regular text)

5Special Features

colspan="3" ‚Üí Makes a cell stretch across 3 columns
rowspan="2" ‚Üí Makes a cell stretch down 2 rows

These are like merging cells in Excel!

üìù Complete Code

<!DOCTYPE html> <html> <head> <title>Student Results</title> </head> <body> <!-- Start of the table --> <table border="1" cellspacing="0"> <!-- Table title --> <caption>Dhaka College</caption> <!-- First row - main heading --> <tr> <th colspan="3">H.S.C Result</th> </tr> <!-- Second row - column headers --> <tr> <th>Roll</th> <th>Name</th> <th>GPA</th> </tr> <!-- Student data rows --> <tr> <td>01</td> <td>Habib</td> <td rowspan="2">5.00</td> </tr> <tr> <td>02</td> <td>Habiba</td> </tr> </table> </body> </html>
